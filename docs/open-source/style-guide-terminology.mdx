some important terms in the Helium Ecosystem

## Glossary

### Hotspot(s)

- always capitalized
- not "gateway"

### 5G Hotspot(s)

- Helium Hotspots capable of connecting to CBRS Radios
- see also "AGW"

---

## Networks and Tokens

### IOT vs IoT (Helium LoRaWAN)

- IOT is the Helium IOT Network Token
- IoT is the Helium IoT Network

### MOBILE vs Mobile (Helium 5G)

- MOBILE is the Helium MOBILE Network Token
- Mobile is the Helium Mobile Network
- Helium 5G is the Helium 5G Network
